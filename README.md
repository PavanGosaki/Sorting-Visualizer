📊 Sorting Algorithm Visualizer

A Python GUI application that visually demonstrates how different sorting algorithms work step-by-step. The application helps users understand the internal working of sorting techniques through animated graphical bars.

The project is built using Python and Tkinter and allows users to generate random data and visualize multiple sorting algorithms with adjustable speed.

🚀 Features
Generate random dataset for sorting
Visualize multiple sorting algorithms
Adjustable sorting speed
Color indicators to show comparisons and swaps
Interactive Tkinter GUI
Helps in learning Data Structures & Algorithms visually
🧠 Sorting Algorithms Implemented

The visualizer supports multiple sorting techniques:

Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
Heap Sort

Each algorithm shows how elements are compared and rearranged during sorting.
